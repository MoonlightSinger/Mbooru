Features

2015/10/07
Mbooru is a moebooru client for Android.
Features:
  • Android 4.0+
  • Support Moebooru & Danbooru & Gelbooru
  • Support browse post, pool, tag, artist
  • Full screen post view
  • Browse comment
  • Browse parent post
  • Download post
  • Vote post
  • Muzei Wallpaper
Support sites:
  • yande.re
  • konachan.com
  • danbooru.donmai.us
  • gelbooru.com
